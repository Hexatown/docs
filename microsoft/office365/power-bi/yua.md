BI energía le jump'éel nu'ukula' informes analítica, self - service.

BI energía u disponible utia'al je'el máaxake' yéetel jump'éel yilaje' Azure Active Directory ku analte'obo' yaan tuláakal le usuarios Office 365. BI energía Cha' u le usuarios importar wa conectar ti' le datos, combinar yéetel jejeláas fuentes datos, ku ts'o'okole' meentik informes ka dashboards. Juntéene' tu, le informes táan u béeytal u compartidos yéetel láak'o'ob yéetel le datos táan u béeytal u actualizados automáticamente.  

BI energía consta Kanp'éel componentes k'ajle': le mayaj BI potencia, potencia BI Desktop, le joonajo' yaan tu datos tu kúuchil ku yéetel le clientes móviles u páajtalil BI. Le mayaj BI energía (app.powerbi.com) u utilizar u ti' u le chíikbesik utia'al meyaj yéetel datos ti' Excel ti', archivos analte'o', wa tu Analysis Services. Ba'axten lo general, informes u crean diseñan usando energía BI Desktop yéetel ku desplegaron ti' le mayaj BI energía. 

Escritorio BI energía meyajtiko'ob le ti' jump'éelili' kuchil-ma'alo'obtal u tasbil u datos (consulta energía) yéetel kisbuuts'o' (Power Pivot) yéetel Excel modelos datos. Modelos datos construidos ti' Excel táan u béeytal u conectados le mayaj yéetel usados wa importados ti' escritorio BI energía. 
Le mayaj BI energía páajtal conectar u ti' le datos dondequiera ku bixake', ku wa ti' le múuyal. Locales tu conexiones u ku beetiko'ob ti' le joonajo' yaan tu datos tu kúuchil ku. Le OPDG xan le utilizado tumen Microsoft Flow yéetel PowerApps seen utia'al xook yéetel ts'íib datos operaciones (energía BI le u chéen xooka'). 

BI energía ti' jump'éel ka'anatako'ob móvil utia'al le k'ajle' plataformas móviles. Le ka'anatako'ob móvil Cha' le máako'obo' tu u muuk' séeba'an u jach yáax utia'al u informes. Informes páajtal diseñar u específicamente utia'al u ka'anatako'ob móvil.
